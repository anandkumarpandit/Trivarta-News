# News Portal

A modern, full-stack news portal built with the MERN stack (MongoDB, Express, React, Node.js).

### 1. Clone the repository
```bash
git clone <your-repo-url>
cd News
```

### 2. Backend Setup
```bash
cd backend
npm install
```

### 3. Frontend Setup
```bash
cd ../frontend
npm install
```
